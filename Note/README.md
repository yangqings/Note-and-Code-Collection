## 学习笔记整理

多动手敲代码、debug，画图、思考、总结、做笔记，最好是问题驱动学习，动手做小项目，做综合性小项目，坚持长期主义。

1. **C、C++、Go、Rust**

   主要是最重要的基础知识和进阶知识，简单的基础知识不再记录。

   *C++*：学习的重点。

   *Go*：静态强类型、编译型、并发型、具有GC的编程语言。语法与C相近。（学习）

   *Rust*：系统编程语言，解决**高并发**和**高安全**系统问题的编程语言。语法与C++相近。（以后再考虑学习）
